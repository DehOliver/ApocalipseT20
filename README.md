# Sistema T20 — Apocalipse

Livro digital estático, criado para publicação no GitHub Pages.

## Publicar no GitHub Pages

1. Crie um repositório no GitHub e envie estes arquivos para a branch `main`.
2. Abra **Settings → Pages** no repositório.
3. Em **Build and deployment**, escolha **Deploy from a branch**, selecione `main` e a pasta `/ (root)`.
4. Salve. O endereço público aparecerá nessa mesma tela após alguns minutos.
