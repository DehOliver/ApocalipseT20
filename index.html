INTRODUÇÃO

Antes de entrar na zona morta

Este sistema acrescenta pressão, escassez e escolhas difíceis às aventuras de T20. O objetivo não é apenas derrotar ameaças: é chegar vivo ao próximo amanhecer.

Informação

Use esta página como modelo para novos capítulos. Cada regra deve ter uma explicação curta, um exemplo e, quando necessário, uma nota para o mestre.

SOBREVIVÊNCIA

Todo recurso acaba

Fome, sede, sono, infecção, moral e tensão tornam cada expedição uma decisão relevante.

Atenção

Não trate a sobrevivência como punição. Ela deve criar escolhas e cenas memoráveis.

COMBATE

Sobreviver é vencer

Armas, munição, mira e cobertura importam. Porém, o barulho pode ser o inimigo mais perigoso.

Regra

Todo disparo gera Barulho. Registre o nível de Barulho da cena e consulte a tabela apropriada quando ele aumentar.

Exemplo

Uma pistola resolve a ameaça imediata, mas pode atrair algo que o grupo não consegue enfrentar. Às vezes, fugir é a melhor tática.

ZUMBIS

A ameaça nunca dorme

Do zumbi comum aos mutantes, cada ameaça pede uma resposta diferente do grupo.

EQUIPAMENTOS

BASES

VEÍCULOS

A estrada pode salvar — ou matar

Carros, motos, caminhões e helicópteros oferecem mobilidade, mas exigem combustível e manutenção.

MESTRE
